# Contributing

Thanks for wanting to contribute! Here's how to get started.

## Setup

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to see the site.

## Making changes

1. Fork the repo and create a branch from `main`.
2. Make your changes.
3. Make sure the build passes: `npm run build`.
4. Open a pull request against `main`.

## Guidelines

- Keep PRs focused — one feature or fix per PR.
- Follow the existing code style (TypeScript, Tailwind, Framer Motion).
- Test your changes locally before opening a PR.
- Blog content and personal copy are not open for contribution.

## Reporting issues

Open an issue if you find a bug or have a suggestion. Include steps to reproduce if it's a bug.
