# `<EPC />`

My personal portfolio — where I pretend to have my life together through clean UI and smooth animations.

## What's inside

- **Hero** — name, tagline, and a sparkle that won't quit
- **Experience** — timeline of places that trusted me with production code
- **Projects** — things I shipped (and a few things that shipped me)
- **Open Source** — PRs that got merged and repos I maintained
- **Skills** — a wall of tech pills that somehow all fit together
- **Blog** — long-form takes on WebRTC, state management, and other things people glaze over at parties
- **Contact** — links to find me (or avoid me, your call)

## Tech stack

- **Next.js 16** + React 19
- **Tailwind CSS 4**
- **Framer Motion** — smooth, GPU-friendly animations
- **Lenis** — smooth scroll, because native scroll is for the weak
- **MDX** — blog posts that mix prose and code

## Run locally

```bash
npm install
npm run dev
```
